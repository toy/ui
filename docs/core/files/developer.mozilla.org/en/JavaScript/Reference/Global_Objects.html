This chapter documents all the JavaScript objects, along with their methods and properties.

The term "global objects" here is not to be confused with the global object. Here, global objects refer to objects in the global scope. The global object itself can be accessed by this in the global scope. In fact, the global scope consistsÂ of the properties of the global object (including inherited properties, if any).
