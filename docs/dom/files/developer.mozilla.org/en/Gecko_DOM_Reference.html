Media related interfaces

These interfaces are used for audio and video media related tasks.

  • HTMLAudioElement, implemented by the HTML5 <audio> element.
  • HTMLVideoElement, implemented by the HTML5 <video> element.
  • NotifyAudioAvailableEvent, describing an event delivered to allow scripts to have access to the audio stream's raw sample data.
