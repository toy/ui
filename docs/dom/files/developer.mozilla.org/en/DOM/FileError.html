Represents an error that occurs while using the FileReader interface.

Note: The values of the error codes listed here are as currently implemented; however, they do not match the latest working draft of the File API specification and will therefore likely change at some point soon.

Properties

Property Type Description
code unsigned short
 The error code.

Error codes

Constant Value Description
NOT_FOUND_ERR 8 File not found.
NOT_READABLE_ERR 24 File could not be read.
SECURITY_ERR 18 The file could not be accessed for security reasons.
ABORT_ERR 20 The file operation was aborted, probably due to a call to the FileReader abort()Â method.
ENCODING_ERR 26 The file data cannot be accurately represented in a data URL.
