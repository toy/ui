HTML Anchor Element Interface

DOM anchor elements expose the HTMLAnchorElement (or HTML 4 HTMLAnchorElement) interface, which provides special properties and methods (beyond the regular element object interface they also have available to them by inheritance) for manipulating the layout and presentation of hyperlink elements.

Properties

Name Type Description
accessKey HTML 4 DOMString A single character that switches input focus to the hyperlink. Obsolete in HTML5
charset HTML 4 DOMString The character encoding of the linked resource. Obsolete in HTML5
coords HTML 4 DOMString Comma-separated list of coordinates. Obsolete in HTML5
hash HTML5 DOMString The fragment identifier (including the leading hash mark (#)), if any, in the referenced URL.
host HTML5 DOMString The hostname and port (if it's not the default port) in the referenced URL.
hostname HTML5 DOMString The hostname in the referenced URL.
href
 DOMString Reflects the href HTML attribute, containing a valid URL of a linked resource.
hreflang
 DOMString Reflects the hreflang HTML attribute, indicating the language of the linked resource.
media HTML5 DOMString Reflects the media HTML attribute, indicating the intended media for the linked resource.
name HTML 4 DOMString Anchor name. Obsolete in HTML5
pathname HTML5 DOMString The path name component, if any, of the referenced URL.
port HTML5 DOMString The port component, if any, of the referenced URL.
protocol HTML5 DOMString The protocol component (including trailing colon (:)), of the referenced URL.
rel DOMString Reflects the rel HTML attribute, specifying the relationship of the target object to the link object.
relList HTML5 readonly DOMTokenList Reflects the rel HTML attribute, as a list of tokens.
rev HTML 4 DOMString Reverse link type. Obsolete in HTML5
search HTML5 DOMString The search element (including leading question mark (?)), if any, of the referenced URL
shape HTML 4 DOMString The shape of the active area. Obsolete in HTML5
tabIndex HTML 4 DOMString The position of the element in the tabbing navigation order for the current document. Obsolete in HTML5
target
 DOMString Reflectst the target HTML attribute, indicating where to display the linked resource.
text HTML5 DOMString Same as the textContent property.
type
 DOMString Reflects the type HTML attribute, indicating the MIME type of the linked resource.

Methods

Name & Arguments Return Description
blur()
 void
 Removes keyboard focus from the current element.
focus()
 void
 Gives keyboard focus to the current element.

Note: These methods are inherited from HTMLElement in HTML5, but defined on HTMLAnchorElement in DOM Level 2 HTML.
