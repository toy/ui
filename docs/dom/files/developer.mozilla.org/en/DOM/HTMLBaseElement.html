HTML Base Element Interface

The base object exposes the HTMLBaseElement (or HTML 4 HTMLBaseElement) interface which contains the base URI for a document.  This object inherits all of the properties and methods as described in the element section.

Properties

Name Type Description
href
 DOMString
 Reflects the href HTML attribute, containing a base URL for relative URLs in the document.
target
 DOMString
 Reflects the target HTML attribute, containing a default target browsing context or frame for elements that do not have a target reference specified.
