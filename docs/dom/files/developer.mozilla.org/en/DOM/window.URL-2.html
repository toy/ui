Introduced in Gecko 2.0

(Firefox 4 / Thunderbird 3.3 / SeaMonkey 2.1)

Summary

Releases an existing object URL which was previously created by calling window.createBlobURL().  Call this method when you've finished using a object URL, in order to let the browser know it doesn't need to keep the reference to the file any longer.

Syntax

objectURL = window.revokeObjectURL(file);
  • file is a File object indicating the file to create a object URL for.
  • objectURL is the generated blog URL. The entire contents of the specified file are represented by the text of the URL.
