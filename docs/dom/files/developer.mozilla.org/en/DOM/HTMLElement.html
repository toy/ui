HTMLElement is an abstract interface for the DOM from which all HTML elements inherit.

attribute DOMString id
attribute DOMString title
attribute DOMString lang
attribute DOMString dir
attribute DOMString className

The following elements exclusively inherit from HTMLElement:

  • special: SUB, SUP, SPAN, BDO
  • font: TT, I, B, U, S, STRIKE, BIG, SMALL
  • phrase: EM, STRONG, DFN, CODE, SAMP, KBD, VAR, CITE, ACRONYM, ABBR
  • list: DD, DT
  • NOFRAMES, NOSCRIPT
  • ADDRESS, CENTER
