HTML Label Element Interface

DOM Label objects inherit all of the properties and methods of DOM element, and also expose the HTMLLabelElement(or HTML 4 HTMLLabelElement) interface.

Properties

Name Description Type
accessKey HTML 4 Reflects the accesskey  HTML attribute. Obsolete  in HTML5 DOMString
control HTML5 The labeled control. Readonly HTMLElement
form
 The form owner of this label. Readonly HTMLFormElement
htmlFor
The ID of the labeled control. Reflects the for attribute. DOMString
